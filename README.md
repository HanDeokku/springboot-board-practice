# Spring Boot 게시판 웹사이트

스프링부트와 타임리프 기반의 간단한 질문과 답변을 할 수 있는 게시판입니다.

점프투스프링 위키독스를 보며 따라 만든 첫번째 웹 프로젝트입니다. 
AWS 라이트세일을 통해 서버를 운용하였습니다.


## 기술 스택
- Spring Boot
- Thymeleaf
- H2 Database
- Spring Data JPA
- Bootstrap
- HTML/CSS
- Javascript
- VSCODE

## 주요 기능
- 질문과 답변
- 댓글
- 좋아요
- 게시판
- 검색
- 회원가입
- 로그인
- 페이징
- 앵커
